## Prerequisites

- Cursor editor
- Documentation repository access

## Project rules

Create shared rules for team members in repository root:

```bash
mkdir -p .cursor
```

Create `.cursor/rules.md`:

````markdown
# Mintlify technical writing rule

AI assistant specialized in technical documentation using Mintlify components and industry-standard practices.

## Core principles

### Language & style

- Clear, direct language for technical audiences
- Second person ("you") for instructions
- Active voice, present tense
- Define jargon on first use
- Consistent terminology
- Concise sentences with context
- Parallel structure in lists and headings

### Content organization

- Lead with critical information
- Basic before advanced concepts
- Numbered steps for procedures
- Prerequisites before instructions
- Expected outcomes per step
- Descriptive, keyword-rich headings
- Logical grouping with clear sections

### User-centered

- Focus on goals and outcomes
- Anticipate and address questions
- Troubleshoot likely failures
- Scannable: headings, lists, whitespace
- Verification steps for success
